2025 年末回顧

Author: 堇姬Naup

今年應該是數一數二充實的一年了XD
我發現在神祕的契機下，認識了蠻多人的
今年同時也是開始接觸 Linux Kernel 的一年，也是開始大學生活的第一年

一月:
2025 年的第一場 CTF 是 TSC CTF，因為當時不是大學生跑去打公開賽區，最後總排名第五
https://naup.mygo.tw/2025/01/17/TSCCTF-2025-writeup/

也大概是一月左右開始接觸 Linux Kernel 的 pwn，因為那時候不用考學測了，有將近半年的空閒，就來亂玩，不過那時候比較多還是在嘗試搞懂作業系統，真的是個大坑XD，不過真的好玩，我 blog 應該也寫過不少篇的筆記可以參考看看

二月:
先跟 CakeisTheFake 打了 AIS3 EOF Final，拿了銀質獎
比起去年第一次打不知道在打甚麼，這次有進入狀況，打得蠻爽的XD
https://naup.mygo.tw/2025/03/27/AIS3-EOF-2025-Final/

然後這個月也跑去 Xpark 看 Tomorin

三月:
跟 OsGa 講了 SITCON 的開放式議程，聽說反響非常不錯，希望有更多人能一起來打一輩子的 CTF(X
https://sitcon.org/2025/agenda/c47604/

然後三月跑去加了竹狐
另外第一次參加 DEVCORE CONFERENCE，相當內容豐富的一場，學到很多

四月:
跟星爆牛炒竹狐打 DEF CON Qual，第一次體會到，要打逆向要逆向，要打 Pwn
要逆向，要打 web 要逆向，要打 crypto 要逆向

五月:
第一次在 My First CTF & pre-exam 出題，出了一些有趣的題目
然後因緣巧合之下在打 BYU CTF 找不到隊友所以發徵隊伍公告，後來在這場就認識了 Bing Chilling Academies 的人，也跟他們破台了這場跟 San Diego CTF 的 Pwn 題，後來就跟他們一起跟 Squid Proxy Lovers 的人認識了就一起打比賽到現在
https://naup.mygo.tw/2025/05/12/San-Diego-CTF-2025-writeup/

六月:
跟星爆牛炒竹弧打 Google CTF，好玩的一場
https://naup.mygo.tw/2025/07/05/2025-Google-CTF-writeup/

這個月也正式的結束了我的高中生活，畢業啦

七月:
第一次在 ISIP 的期末發發表分享，講了我在打 CTF 的時候，通過 initrd 的神秘特性，成功用一個巧妙的方法 get root
另外在這個月分加入了 Deep Hacking，Deep Hacking 讀書會的內容真的能學到非常多東西，非常好玩
另外也打了 Crypto CTF，都在 coppersmith，然後拿了第 23 名，好玩
https://naup.mygo.tw/2025/07/20/2025-cryptoCTF-writeup/

在這個月跟 Normal Subgroup 的人熟了起來，後來他們有比賽有時也會邀我線上打，蠻多都蠻好玩的

這個月也跟 SPL 的人打了 DownUnder CTF，第一次在賽中自己解出 Linux Kernel 的題目，這場題目也蠻有趣的，最後拿了第 8
https://naup.mygo.tw/2025/07/20/2025-downunderCTF-writeup/

另外也第一次擔任了 AIS3 營隊的 CTF 組助教，跟其他助教挑戰在一周內 speed run 出 KoH 跟 A&D 的題目，也是我第一次出這類型的題目，希望大家都能玩的開心XD
https://naup.mygo.tw/2025/08/04/AIS3-2025-KoH-and-TA/

八月:
今年一樣去參加了 HITCON
另外也去 AIS3 Junior 當助教，順便在台南吃了好多東西XD

今年也打了 HITCON CTF 解了非常多題目，能感受到自己的進步，也順利進入決賽~
https://naup.mygo.tw/2025/10/24/2025-HITCON-CTF-Writeup/

另外八月也去 CTF 資安種子培訓交流會講了一個議程，這邊再次感謝資通院的邀請

九月:
開始交大的第一個學期，猶豫了一陣子還是跑去修了編譯器，好課
另外跟 CakeisTheFake 打了神盾杯，終於升大學了可以打這場了，初賽拿了第 3 名，順利進入決賽
https://naup.mygo.tw/2025/10/24/2025-AEGIS-CTF-writeup/

另外 ISIP 今年新的一屆是第一次參加導師培訓，導師是 pumpkin，今年非常感謝 pumpkin 的指導，也幫我解惑了許多問題跟給了我許多方向

十月:
跟竹狐打 HITCON CTF Final (應該叫做交流賽就是了)，在 Belluminar 出了一題自己針對 interrupt 的研究題目，順便說這場也是我今年出得第三題 0 solved，QQ
這場相當好玩第一次打 Belluminar 這個賽制，也是第一次打 LiveCTF
最後拿了冠軍

在我打 HITCON CTF Final 時候，剛好是金盾初賽，強大的隊友單刷帶我進了決賽，感謝隊友XD

今年也繼續挑戰了 FlareOn CTF，又是每年不是 reverser 但跑去玩的時候，去年解了 5/10，今年則是 8/9，只差了最後一題，不過 FlareOn 是真的蠻好玩的XD

另外也跟 SPL 打了 Osu Gaming CTF，拿了冠軍，這是我第一次在 CTFtime 比賽上拿了冠軍，也拿到了第一次在 CTFtime 比賽上的獎品，一件 OSU 的 T-shirt

另外也跟 Arr3stY0u 打了 XCTF 的決賽，那時候遠端支援了 A&D 的 pwn 題目，最後拿了第六名，好玩

十一月:
跟 CakeisTheFake 打了神盾杯的決賽，主要負責打 Pwn 的 A&D 題目，另外一邊我不會滲透所以交給隊友 carry，最後拿下了第二名，好玩
慶功宴去吃了神秘的串燒店

另外也拿到了 DEVCORE 獎學金，非常感謝 DEVCORE，這是對我一個非常大的鼓勵

十二月:
第一次在 DeepHacking 發表，講了延續我在 HITCON CTF 交流賽出的題目，並根據這招來針對 Dirty PageTable 提出了一些好玩的 exploit 方式

另外跑去出了 AIS3 EOF 的題目，希望大家都可以成為魔法少女和寶可夢大師

以上就是今年的年末回顧，今年玩了非常多好玩的東西，期望明年能玩更多有趣的事情，能體驗更多不同的東西
