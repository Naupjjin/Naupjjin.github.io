CPU暫存器

隨設計而改變用途

名稱 意思
rax accumulator
rbx base
rcx count
rdx data

固定用途

名稱 意思
rsi source index
rdi destination index
rbp base pointer(控制stack)
rsp stack pointer(控制stack)
rip instruction pointer(執行到甚麼位置)
  • 同個暫存器可以用不同名稱來存取不同大小資料(RAX/EAX/AX/AH/AL是同個暫存器，只是大小不同)

ex:
RAX= 0x0102030405060708
EAX= 0x05060708
AH = 0x0506
AL = 0x0708
